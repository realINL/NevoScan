PRIVACY POLICYOF THE MOBILE APPLICATION“NevoScan”Effective date: 01.05.2026Version: 1.01. General Provisions1.1. This Privacy Policy (hereinafter referred to as the “Policy”) defines the procedure for data processing when using the NevoScan mobile application (hereinafter referred to as the “Application”).1.2. The Application is intended for preliminary preventive analysis of a mole photograph using machine learning algorithms.1.3. Upon the first launch of the Application, the user must read this Policy and confirm their consent to its terms. Without such consent, the analysis functions of the Application are unavailable.1.4. Confirmation of consent means that the user voluntarily agrees to the processing of a mole photograph under the terms specified in this Policy.2. Nature of the Application2.1. The Application is a preventive tool and is not a medical device.2.2. The analysis results are provided exclusively for informational and reference purposes. The Application does not make medical diagnoses, prescribe treatment, or replace consultation with a qualified medical professional.2.3. In case of any doubts, changes in a mole, complaints, or other signs of risk, the user should consult a dermatologist.3. Data Processed by the Application3.1. The Application requests only a photograph of a mole selected or taken by the user. The Application does not request any other personal data, including name, phone number, email address, passport details, or medical record data.3.2. The photograph is processed solely for the purpose of performing the analysis and displaying the result to the user.3.3. Since the Application does not require the user to provide personal data, the processed information is not linked to a specific individual and is considered anonymized within the operation of the Application.4. Local Data Processing and Storage4.1. In the mobile application, the photograph analysis is performed locally on the user’s device.4.2. Photographs, analysis results, and related data are stored only on the user’s device and are not transferred to the Operator’s servers, third-party services, medical organizations, or any other third parties.4.3. The Operator does not have remote access to photographs or analysis results stored in the Application on the user’s device.4.4. The user deletes the data independently using the tools of the Application or the operating system of the device.5. Data Transfer to Third Parties5.1. The Application does not transfer photographs of moles, analysis results, or other related data to third parties.5.2. The data is not used for model training, advertising purposes, user behavior analytics, or transfer to medical institutions.5.3. Data transfer is possible only if the user independently exports, sends, or otherwise transfers the data outside the Application.6. Data Security6.1. The security of data stored on the device depends on the user’s device settings, including the use of a password, biometric protection, screen lock, and up-to-date operating system updates.6.2. Users are advised to use standard device security tools and not to provide access to the device to third parties.6.3. When the Application is deleted, the data stored in it may be deleted in accordance with the rules of the device’s operating system.7. User Consent7.1. Upon the first launch of the Application, the user confirms their consent to this Policy through the consent screen.7.2. The user has the right to refuse to use the Application if they do not agree with the terms of this Policy.8. Final Provisions8.1. The Operator has the right to make changes to this Policy. The new version of the Policy comes into force from the moment it is published in the Application or on the official website, unless otherwise specified in the new version.8.2.
